************************
Preliminaries
************************


In this chapter, we will discuss the collection of software tools used in
developing and deploying robots.  Most of these tools are not robotics
specific such as Python or ZeroMQ.   Others like ROS have been developed
just for robotics.    We begin with an overview of the specific robotics
tools.  Next we will cover so basic concepts in operating
systems architecture and software design.  The core tools we will employ,
ZeroMQ and SciPy are introduced.
